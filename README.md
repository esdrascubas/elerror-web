# Proyecto Web: Logo de puntos interactivo

- `index.html` es la base.
- Render en canvas desacoplado en `js/lib/dotsRenderer.js`.
- Mapa (JSON) parametrizable: por defecto `assets/map.json`, o `?map=assets/otro.json`.
- Colores configurables por CSS en `css/styles.css` (`--bg-color`, `--logo-color`).
